What we audit
Seven audit categories
Each finding maps to a violation type — billing, efficiency, or execution. Every violation carries a dollar amount.
Anomaly detection
Token usage spikes, unexplained cost jumps, spend pattern outliers across models and services.
Billing
Token usage audit
Input / output / cache token breakdown — how you are actually billed vs what was consumed.

SLA uptime

How we collect data
Two data streams. One reconciliation.
Usage data is our independent ledger — what actually happened. Billing data is what the vendor says you owe. The gap between them is where the money is.
Usage data — our independent ledger
SDK wrapper (Y)
request_id, model, input/output/cache tokens, duration, tool calls, retries, errors. Per-call metadata. Prompt text never sent by default.
Cloud cost APIs (Tun)
Azure Cost Management · AWS Cost Explorer · GCP Billing API. Structured billing data — richest source for SLA breach detection.
Admin APIs (Tun)
ChatGPT Ent admin · Claude.ai Ent admin · M365 Graph — seat activity and utilisation rates.
CLI telemetry (Nayana)
GitHub org usage API · Cursor admin CSV export · Claude Code admin API. Dev tool seat and acceptance metrics.
Billing data — what the vendor says you owe
Provider usage exports
OpenAI usage CSV · Anthropic console export · GCP billing export for Gemini. What the vendor reports as your consumption.
Customer invoice upload
PDF / CSV invoices. We extract line items and cross-reference against our independent ledger. Every delta is a finding.
Pricing tables
Vendor rate cards · per-model pricing · committed-use discounts · PTU rates. Applied at log time for point-in-time accuracy.
SLA commitments
Uptime guarantees · latency SLAs · enterprise agreement terms. The contractual basis for credit claims.

Reconciliation = usage ledger vs billing data

Compare what we independently recorded against what the vendor says you owe. Every mismatch is a finding. Every finding has a dollar amount. Mark owns the claim workflow — evidence packaging, vendor submission, resolution tracking.
